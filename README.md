# MADDE
> **M**ake **A**I **D**riven **D**evelopment **E**asier.

**MADDE** is a curated toolkit, workflow collection, and configuration set designed to minimize friction in AI-assisted software development.

The name comes from the Turkish word *"Madde"* (meaning **"Matter"** or **"Substance"**).
In the context of AI coding, **Context is the Matter.** This repository aims to provide the essential "matter" to your LLMs in the fastest, most efficient way possible.

To bridge the gap between your local development environment and AI models. We focus on:

- Sharing proven prompt engineering workflows.
